## 🚀 使用方法

### 1. ローカル環境での実行

1. ファイルをダウンロード
2. `index.html` をブラウザで開く
3. すぐに動作確認が可能

### 2. Webサーバーへのデプロイ

1. すべてのファイルをサーバーにアップロード
2. `index.html` がルートディレクトリにあることを確認
3. ブラウザでアクセス

### 3. カスタマイズ方法

#### 色の変更
`css/style.css` の冒頭にある CSS Variables を編集:

```css
:root {
    --primary-color: #9d8b6c;  /* メインカラー */
    --primary-dark: #7d6b4c;   /* ダークカラー */
    --primary-light: #c1a87d;  /* ライトカラー */
    /* ... */
}
```

#### 画像の追加
1. 画像ファイルを `images/` フォルダに保存（新規作成）
2. HTMLの `.image-placeholder` 部分を以下のように変更:

```html
<!-- 変更前 -->
<div class="image-placeholder">
    <i class="fas fa-spa"></i>
    <p>Image</p>
</div>

<!-- 変更後 -->
<img src="images/your-image.jpg" alt="説明文">
```

#### コース情報の変更
`index.html` の Courses Section を編集してください。
